# Description:
A financial tool designed to track and manage stock investments in real-time. This project highlights the ability to work with Object-Oriented Programming (OOP) and integrate third-party APIs for live data.
# Key technical features:
Live API Integration: Leverages the yfinance library to fetch current market prices directly from Yahoo Finance.

OOP Architecture: Built using a StockPortfolio class to encapsulate data and methods for adding, removing, and valuing assets.

Real-time Calculations: Automatically calculates the total portfolio value based on the current market price of each ticker and the quantity held.
